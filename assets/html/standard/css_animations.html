Important Note

All the attributes of the CSS animations can be manipulated through a data-[CSS animation property] attribute which can be added directly into the HTML element. The system will add for you the prefix of the CSS property automatically (if any).

For Example:

<button class="evo_h-anim--fadeIn" data-animation-duration="5s">Test</button>

<button class="evo_h-anim--fadeIn" data-animation-duration="2s" data-animation-timing-function="ease-out">Test</button>
